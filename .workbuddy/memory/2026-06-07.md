### 关键技术点
- ICO 文件中 32bpp DIB 格式的图像以 BGRA 存储，需转换为 RGBA
- CMake 中修改 exe 输出名不能用 `set(BINARY_NAME)`（会破坏 flutter run），应用 `set_target_properties OUTPUT_NAME`
- Node.js zlib 可以生成和解析 PNG 文件（无需 Pillow/Sharp 等外部依赖）
- PNG 文件结构：IHDR → IDAT（zlib 压缩的像素数据）→ IEND

## 安装包修复：7-Zip SFX 替换 BAT 存根 (11:17 - 11:22)

### 问题
- 之前的 `盘古影视_Setup.exe` 是用 `cat installer_stub.bat + package.zip` 生成的
- 文件头是 `@echo off`（BAT 脚本），不是 `MZ`（PE 可执行文件）
- Windows 双击 .exe 时按 PE 格式加载，BAT 内容无效 → "此应用无法在你的电脑上运行"

### 解决方案
- 使用 7-Zip SFX 模块 (`C:\Program Files\7-Zip\7z.sfx`) 创建真正的 PE 自解压安装包
- `7z.sfx` 是真正的 PE 可执行文件（文件头 `MZ`），Windows 可正常加载
- SFX 配置文件：`sfx_config.txt`（UTF-8，含安装标题、安装路径、桌面快捷方式、自动运行）
- 打包命令：
  1. `7z a -mx=9 package.7z Release/*` → 26MB（solid LZMA2）
  2. `cat 7z.sfx sfx_config.txt package.7z > 盘古影视_Setup.exe` → 27MB
- 验证：`xxd` 确认文件头 `4d5a`（MZ），`7z t` 验证内嵌存档完整（29 文件通过）

### SFX 配置关键项
- `InstallPath="C:\\Program Files\\盘古影视"`
- `RunProgram="盘古影视.exe"`（安装完自动启动）
- `Shortcut="Du,..."`（创建桌面快捷方式）
- `GUIFlags="8+16+32+64+128+256"`（所有必要的 GUI 选项）

### 安装包产出
- `E:\pangu\release\盘古影视_Setup.exe` (27MB, 真正的 PE 自解压安装包)

### 实现步骤
1. **环境准备**：Python 3.14 (64-bit) + Python 3.14-32 (32-bit) + PyInstaller
2. **打包 Release 目录**：`7z a -mx=9 app_data.zip Release/*` → 36MB
3. **重写 installer.py**：
   - 修复 `create_shortcut_ctypes` 中误用 `win32com` 的 bug
   - 改用 PowerShell 创建桌面快捷方式（可靠且无需额外依赖）

   - 64 位：`py -3.14 -m PyInstaller --add-data "app_data.zip;." --icon=app_icon.ico`
   - 32 位：`py -3.14-32 -m PyInstaller --add-data "app_data.zip;." --icon=app_icon.ico`
   - **关键**：必须使用对应架构的 Python 运行 PyInstaller，仅 `--target-architecture` 参数无效

### 安装器功能
- 图形化安装向导（Tkinter GUI，深色主题）
- 可自定义安装路径（默认 `C:\Program Files\盘古影视`）
- 解压内嵌的 `app_data.zip` 到目标目录
- 创建桌面快捷方式（通过 PowerShell）

| 文件 | 大小 | 架构 | 说明 |
|------|------|------|------|
| `盘古影视安装器_32位.exe` | 47MB | PE32 (i386) | 32 位，兼容所有 Windows |
| `盘古影视安装器_64位.exe` | 49MB | PE32+ (x86-64) | 64 位，仅 64 位 Windows |

### 技术要点
- `app_data.zip` 通过 `--add-data` 嵌入，运行时通过 `sys._MEIPASS` 访问

- 32 位 Python 必须使用 `py -3.14-32` 启动器，不能用 64 位 Python 的 `--target-architecture`
